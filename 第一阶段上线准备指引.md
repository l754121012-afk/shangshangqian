# 今日上上签 · 第一阶段上线准备指引

## 一、注册微信小程序账号（你需要亲自操作）

### 1.1 注册入口
- 打开 https://mp.weixin.qq.com/ ，点击"立即注册"
- 选择"小程序"账号类型
- 用一个未绑定过公众号的邮箱注册

### 1.2 账号信息填写
- 账号名称：今日上上签（可后续修改）
- 服务类目：工具 → 效率（或 生活服务 → 综合）
  - 如果"效率"不可选，备选：生活服务 → 综合信息服务
  - 避免选择"社交"、"社区"类目（审核严格）

### 1.3 完善信息
- 登录小程序管理后台
- 开发管理 → 开发设置 → 记录你的 AppID
- 将 AppID 填入 weapp-demo/project.config.json 的 appid 字段

### 1.4 下载微信开发者工具（如已安装跳过）
- https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 选择 Stable 稳定版

## 二、准备内容库

### 2.1 签卡JSON字段定义
每张签卡需要包含以下字段：
```json
{
  "id": "fortune-door",
  "grade": "大吉签",
  "type": "good",
  "seal": "旺",
  "name": "财气临门",
  "text": "今天适合把一个正在犹豫的机会推进半步。",
  "do": "主动推进",
  "avoid": "空想暴富",
  "action": "发出一条能推动事情的小消息。",
  "score": 92
}
```

字段说明：
- id: 唯一标识（英文+连字符）
- grade: 签等级（大吉签/上上签/上签/平签/解签）
- type: 类型（good=正面 normal=普通 low=低迷 negative=负面）
- seal: 印章字（单字）
- name: 签名（2-4字，有力量感）
- text: 签文解读（20-30字，具体可操作的建议）
- do: 宜（2-4字）
- avoid: 忌（2-4字）
- action: 微行动（15-25字，今天就能做的一件小事）
- score: 好运值（大吉签80-100，上上签60-80，上签40-60，平签20-40，解签10-20）

### 2.2 内容分布建议（20张）
- 大吉签 × 4 张（好运值 85-100）
- 上上签 × 5 张（好运值 60-80）
- 上签 × 4 张（好运值 40-60）
- 平签 × 4 张（好运值 20-40）
- 解签 × 3 张（好运值 10-20）

当前Demo中已内置20张完整签卡，可直接用于测试。

### 2.3 内容原则
- 所有签卡**不承诺真实财富、命运改变、灾难消除**
- "转运"类表达全部解释为"换一种视角看问题"
- 负面签必须附带消解方式（"我已克服"按钮）
- 微行动必须是当天就能完成的具体小事
- 广告/盈利入口不得出现在负面签或焦虑情绪场景中

## 三、隐私协议与用户协议

### 3.1 隐私协议页面
需要在小程序中添加一个独立页面展示，内容要点：
- 收集哪些信息（设备信息、使用数据）
- 数据如何使用（改善服务、个性化推荐）
- 不会向第三方出售用户数据
- 用户有权删除自己的数据

### 3.2 用户协议页面
- 服务说明（娱乐+自我反思工具，非命理预测）
- 免责声明
- 知识产权声明

### 3.3 配置方式
在 app.json 的 pages 数组中添加：
```json
"pages": [
  "pages/index/index",
  "pages/agreement/privacy",
  "pages/agreement/terms"
]
```

## 四、广告组件接入准备（代码先行，达标后启用）

### 4.1 激励视频广告
开通条件：小程序累计UV >= 1000

接入方式（代码提前写好）：
```javascript
// 在 onLoad 中创建广告实例
const rewardedVideoAd = wx.createRewardedVideoAd({
  adUnitId: 'your-ad-unit-id' // 后台创建广告位后替换
})
// 触发广告
rewardedVideoAd.show().catch(() => {
  rewardedVideoAd.load().then(() => rewardedVideoAd.show())
})
// 监听关闭（判断是否完整观看）
rewardedVideoAd.onClose(res => {
  if (res && res.isEnded) {
    // 完整观看，发放奖励
    this.performDraw()
  } else {
    wx.showToast({ title: '需完整观看才能获得奖励', icon: 'none' })
  }
})
```

### 4.2 Banner广告
- 位置建议：牌册页面底部、好友比手气页面底部
- 不放在抽签结果页（影响核心体验）

### 4.3 当前Demo中的广告位置汇总
| 位置 | 类型 | 触发条件 |
|------|------|----------|
| 看广告补充签 | 激励视频 | 免费签用完后 |
| 看广告转运 | 激励视频 | 抽到低迷/负面签 |
| 牌册页底部 | Banner | 进入牌册时 |
| 好友比手气页底部 | Banner | 进入好友页时 |

## 五、上线检查清单

### 提审前必检
- [ ] 所有页面功能正常，无白屏/报错
- [ ] 隐私协议和用户协议页面已配置
- [ ] app.json 中 pages 路径正确
- [ ] 无 console.log 等调试代码
- [ ] 无测试用按钮/文案残留
- [ ] 小程序基本信息填写完整
- [ ] 类目资质审核通过

### 审核期间
- 正常审核1-7个工作日
- 如被驳回，根据驳回原因修改后重新提交
- 不要频繁提交（会影响审核优先级）
