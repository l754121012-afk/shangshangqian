# 今日上上签 H5 Demo

这是一个手机浏览器可用的交互 Demo，用于验证第二版财运风 v2 的核心体验。

## 已包含

- 抽卡与 SSR 好运牌火焰特效
- 今日结果页、宜忌、微行动
- 验证成功后的隐晦感谢入口
- 牌册收集、稀有度、套装解读提示
- 好友比手气与好运火种池
- 模拟广告转运
- 负面牌输入“我已克服 / 我已解决 / 我已放下”后合并消失
- 分享文案复制

## 运行

在 `demo-h5` 目录下启动静态服务器：

```powershell
python -m http.server 5173
```

然后用浏览器打开：

```text
http://localhost:5173/
```

局域网手机预览时，把 `localhost` 换成电脑的局域网 IP。


## 抽签次数与广告测试

- 每日免费 3 抽。
- 第 1 抽是今日主签，第 2-3 抽是补充角度。
- 第 4 抽开始触发 2.6 秒模拟广告。
- 低运 / 负面牌可触发模拟广告转运。
- 验证成功后可测试“添一束好运火”的轻支持入口。
