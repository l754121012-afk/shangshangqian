# 小程序分享机制说明 · 各平台操作指南

## 一、微信小程序原生分享

### 微信群/好友
- **方式**：小程序内点击右上角「···」→「转发」，或页面内自定义分享按钮调用 `wx.shareAppMessage()`
- **效果**：分享卡片形式，带小程序Logo、标题、缩略图
- **限制**：只能分享到微信生态（单聊/群聊/朋友圈不支持直接分享小程序卡片）

### 朋友圈
- **小程序卡片不能直接分享到朋友圈**
- **替代方案**：生成分享海报图片（PNG）→ 保存到相册 → 用户手动发朋友圈
- **海报内容**：签卡截图 + 小程序码 + 文案

## 二、小红书

### 小程序能否直接分享到小红书？
**不能。** 微信小程序没有开放分享到小红书的API。

### 正确操作方式
1. 小程序内生成分享海报（或签卡截图）
2. 用户点击「保存到相册」
3. 打开小红书App → 发布笔记 → 从相册选择保存的图片
4. 复制小红书文案粘贴到正文

### 素材准备清单
- [ ] 签卡截图（3:4比例，小红书最佳）
- [ ] 分享卡片截图（带小程序码）
- [ ] 场景配图（工位/书桌/床头+签卡叠加）
- [ ] 小红书文案（已准备30条）

## 三、抖音

### 小程序能否直接分享到抖音？
**不能。** 微信小程序没有开放分享到抖音的API。

### 正确操作方式
1. 小程序内生成分享海报或录屏
2. 保存到相册
3. 打开抖音App → 发布视频/图文 → 从相册导入
4. 粘贴抖音文案和标签

### 素材准备清单
- [ ] 签卡截图（9:16比例，抖音最佳）
- [ ] 抽卡录屏素材（15-20秒）
- [ ] 场景视频（工位抽卡/分享瞬间）
- [ ] 抖音脚本（已准备4个）

## 四、公众号

### 小程序与公众号的关系
- 公众号文章可以**嵌入小程序卡片**（文字链接/图片链接/小程序卡片组件）
- 用户在公众号文章内点击可直接跳转小程序

### 正确操作方式
1. 在公众号后台编辑器中插入「小程序」组件
2. 搜索「今日上上签」并关联
3. 选择展示方式：卡片/图片/文字链接
4. 发布文章

### 素材准备清单
- [ ] 公众号文章（已准备4篇）
- [ ] 文章配图（签卡截图/分享卡片）
- [ ] 小程序卡片配置（在公众号后台完成）

## 六、小程序端需要实现的分享功能

### 必须实现
1. **生成分享海报**：Canvas绘制签卡+背景+小程序码，保存到相册
2. **微信好友分享**：调用 `wx.shareAppMessage()` 分享小程序卡片
3. **保存签卡截图**：单独保存当前签卡为图片

### 建议实现
4. **一键复制文案**：复制当前签文的文案到剪贴板，方便用户粘贴到小红书/抖音
5. **分享模板选择**：让用户选择7套模板中的一套生成海报
6. **排行榜分享**：分享好友榜截图到微信群
