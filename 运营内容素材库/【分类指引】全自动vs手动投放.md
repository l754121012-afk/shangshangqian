# 运营素材分类指引 · 全自动 vs 手动投放

> 目标：你打开这个文件夹，一眼就知道哪些是程序自动跑的，哪些需要你手动去发。

---

## 全自动（已接入小程序代码，无需你操作）

| 素材 | 位置 | 触发方式 | 你的操作 |
|---|---|---|---|
| 每日金句 | `daily-quotes.json` | 用户每天打开小程序，首页自动展示当天金句 | 无需操作，每月更新JSON即可 |
| 节日专题签卡 | `holiday-events.json` | 到节日日期自动切换主题和专属签卡 | 无需操作，节日前2周检查配置即可 |
| 分享卡片模板 | `share-templates.json` | 用户抽卡后点击"生成分享卡"，自选模板 | 无需操作，新增模板时更新JSON |
| 签卡池/概率系统 | `index.js` | 每次抽卡自动计算概率，自动分配签卡 | 无需操作，调概率时改代码 |
| 称号系统 | `index.js` + `index.wxml` | 集齐签卡自动解锁，自动展示 | 无需操作 |

**结论**：全自动素材 = 你一个月只需要检查一次，确认运行正常即可。

---

## 需要你手动投放的（进这个文件夹直接拿）

| 素材 | 位置 | 平台 | 频率 | 拿取方式 |
|---|---|---|---|---|
| 微信群早安文案 | `外部投放内容/微信群每日早安文案.json` | 微信群 | 每天1条 | 打开JSON，按day顺序复制 |
| 小红书文案 | `外部投放内容/小红书文案.json` | 小红书 | 每天1条 | 打开JSON，按day顺序复制 |
| 公众号文章 | `外部投放内容/公众号文章.json` | 公众号 | 每周1篇 | 打开JSON，按publishWeek复制 |
| 抖音视频脚本 | `外部投放内容/抖音视频脚本.json` | 抖音/视频号 | 每周1条 | 打开JSON，按脚本拍摄 |

**结论**：手动投放素材 = 你每天/每周从这个文件夹复制内容，发到对应平台。

## 快速取用路径

```
运营内容素材库/
├── 全自动（不用管）/
│   ├── daily-quotes.json          ← 每日金句
│   ├── holiday-events.json        ← 节日专题
│   └── share-templates.json       ← 分享模板
│
└── 手动投放（每天来拿）/
    ├── 微信群每日早安文案.json     ← 每天复制1条到微信群
    ├── 小红书文案.json             ← 每天复制1条到小红书
    ├── 公众号文章.json             ← 每周复制1篇到公众号
    ├── 抖音视频脚本.json           ← 每周按脚本拍1条
    └── 使用指南.md                 ← 不知道怎么发就看这个
```

## 月度工作流程

```
每月1号：
  1. 检查全自动素材是否需要更新（金句/节日/模板）
  2. 打开"手动投放"文件夹，确认本月内容已就位
  3. 开始按日历执行投放

每天早8:00：
  1. 打开 微信群每日早安文案.json
  2. 复制当天文案 → 发到微信群

每天午12:00或晚20:00：
  1. 打开 小红书文案.json
  2. 复制当天文案 → 配1-3张图 → 发小红书

每周一晚上：
  1. 打开 公众号文章.json
  2. 复制当周文章 → 排版 → 发公众号

每周三晚上：
  1. 打开 抖音视频脚本.json
  2. 按脚本拍摄 → 剪辑 → 发抖音
```
