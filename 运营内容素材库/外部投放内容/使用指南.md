# 外部投放内容使用指南

## 一、内容分发节奏（1个月）

| 日期 | 微信群 | 小红书 | 公众号 | 抖音 |
|---|---|---|---|---|
| 每天 | 早安文案1条 | 文案1条 | - | - |
| 每周一 | - | - | 文章1篇 | - |
| 每周三 | - | - | - | 视频1条 |

## 二、各平台发布要点

### 微信群
- 时间：早上 8:00-8:30
- 形式：文案 + 分享卡片截图（可选）
- 频率：每天1条，30天不重样
- 互动：偶尔在群里发起"今天谁抽到大吉签"的接龙

### 小红书
- 时间：中午 12:00-13:00 或晚上 20:00-21:00
- 形式：文案 + 1-3张图 + 标签
- 频率：每天1条，30天不重样
- 互动：评论区置顶小程序入口，回复每条评论

### 公众号
- 时间：每周一晚上 20:00
- 形式：长图文，1500-2500字
- 频率：每周1篇，共4篇
- 互动：文末投票"你最喜欢哪张签卡"

### 抖音/视频号
- 时间：每周三晚上 18:00-19:00
- 形式：15-30秒短视频
- 频率：每周1条，共4条
- 互动：评论区置顶"微信小程序搜今日上上签"

## 三、内容复用技巧

1. **一鱼多吃**：同一主题内容可以改编成多平台版本
   - 例：公众号文章《我集齐了所有大吉签》→ 小红书精简版 → 抖音视频版

2. **热点借势**：遇到社会热点时，快速挑选相关签卡做内容
   - 例：高考期间推"金榜签"相关内容

3. **用户UGC**：鼓励用户分享抽卡截图，收集后二次传播
   - 在小红书发起"#今日上上签打卡"话题

## 四、数据追踪

建议记录以下数据：
- 各平台发布内容的点赞/评论/转发数
- 小程序当日新增用户数
- 分享率最高的内容类型
- 转化率最高的CTA文案

每月复盘一次，优化下月内容方向。
